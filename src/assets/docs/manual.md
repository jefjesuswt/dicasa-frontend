# Manual de Usuario - Dicasa Dashboard

Bienvenido al panel de administración de Dicasa. Este manual describe las funcionalidades disponibles para gestionar el inventario inmobiliario, usuarios y agenda.

## 1. Propiedades

La sección de **Propiedades** es el corazón del sistema. Aquí puedes ver, crear, editar y eliminar inmuebles.

### Listado de Propiedades

- Visualiza todas las propiedades en formato de tarjeta.
- Usa la barra de búsqueda para filtrar por título o ubicación.
- Los filtros avanzados permiten refinar por precio, tipo (venta/alquiler) y características.

### Crear/Editar Propiedad

- Haz clic en el botón "Nueva Propiedad" para añadir un inmueble.
- Completa el formulario con la información requerida:
  - **Datos Básicos**: Título, descripción, precio.
  - **Ubicación**: Dirección, ciudad, estado.
  - **Características**: Habitaciones, baños, garage, área.
  - **Imágenes**: Sube fotos de alta calidad.

## 2. Usuarios (Administradores / Gerentes)

Gestión del equipo y clientes del sistema.

- **Ver Usuarios**: Lista completa con roles y estado.
- **Roles**:
  - _Admin_: Acceso total.
  - _Gerente_: Gestión de propiedades y agenda.
  - _Agente_: Solo visualización asignada.

## 3. Agenda

Gestiona las citas y visitas a las propiedades.

- **Calendario**: Vista mensual/semanal de citas.
- **Nueva Cita**: Registra una visita vinculando un cliente y una propiedad.
- **Estados**: Pendiente, Confirmada, Completada, Cancelada.

## 4. Estadísticas (Administradores)

Visualiza el rendimiento del negocio.

- **KPIs**: Total de ventas, propiedades activas, nuevos clientes.
- **Gráficos**: Tendencias de precios y visitas mensuales.
